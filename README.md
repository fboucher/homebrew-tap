# Fboucher Tap

## How do I install these formulae?

`brew install fboucher/tap/<formula>`

Or `brew tap fboucher/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "fboucher/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
